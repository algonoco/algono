# >_ algono

**Least Privileged** — open-source Active Directory audit engine for sysadmins.

## What it is

Algono scans your AD environment, maps privilege exposure across every account and group, calculates blast radii, and compiles a typographically precise PDF report — so you can see exactly what attackers already know.

## The Speakeasy

The scanner is free and open source. Access is gated intentionally. Pick a door:

### Door A — Email Knock
Send a blank email to **knock@algono.co**. The autoresponder replies with the repository link.

### Door B — GitHub Gate
Open an issue titled `init` on [algonoco/access](https://github.com/algonoco/access). A GitHub Action comments the link and closes the issue.
